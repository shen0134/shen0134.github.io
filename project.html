Project 4: Joke Generator Page

This project involved creating a webpage that generates a random joke every time a button is clicked. The main feature of the page is a button that, when pressed, uses JavaScript to manipulate the DOM and fetch a new joke from an array of pre-defined jokes. This project demonstrates basic JavaScript skills, including DOM manipulation and event handling, to create a dynamic and interactive user experience.

Technologies Used:

  • HTML: For structuring the page layout.
  • CSS: For styling the page.
  • JavaScript: For event handling,and joke generation logic.
